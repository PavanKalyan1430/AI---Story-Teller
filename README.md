## Installation

No external dependencies required! Just Python 3.6 or higher.

```bash
git clone https://github.com/PavanKalyan1430/AI---Story-Teller.git
cd AI---Story-Teller
```

## Usage

### Interactive Mode

Run the story teller in interactive mode:

```bash
python3 story_teller.py
```

### Example Interaction

```
You: hey hi
AI Story Teller: Hello! I'm your AI Story Teller. What kind of story would you like to hear?

You: tell me an adventure story
AI Story Teller: Once upon a time, a brave knight set out on a journey to the Enchanted Forest...

You: quit
AI Story Teller: Goodbye! Come back soon for more stories!
```

### Command Examples

- **Greetings**: `hi`, `hello`, `hey`, `greetings`
- **Story Requests**: 
  - `tell me a story`
  - `I want an adventure story`
  - `tell me a mystery`
  - `I want a fantasy story`
- **Exit**: `quit`, `exit`, `bye`

## Running Tests

Run the test suite to verify functionality:

```bash
python3 -m unittest test_story_teller.py -v
```
